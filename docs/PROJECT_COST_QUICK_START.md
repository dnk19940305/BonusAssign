# 项目成本管理 - 快速开始指南

## 🚀 5分钟上手指南

### 步骤1: 录入第一条成本记录

1. 访问 **项目成本管理** 页面
2. 点击右上角 **"录入成本"** 按钮
3. 填写表单：
   ```
   项目：选择您的项目
   成本类型：选择"人力成本"
   金额：150000
   发生日期：选择今天
   状态：已确认
   描述：开发团队工资支出（第一季度）
   ```
4. 点击 **"创建"** 按钮
5. ✅ 看到成功提示并自动刷新列表

### 步骤2: 编辑成本记录

1. 在成本列表中找到刚创建的记录
2. 点击该记录行的 **"编辑"** 按钮
3. 修改金额为 `160000`
4. 修改描述为 `开发团队工资支出（第一季度，含绩效奖金）`
5. 点击 **"更新"** 按钮
6. ✅ 看到"成本记录更新成功"提示

### 步骤3: 删除成本记录

1. 点击记录行的 **"删除"** 按钮
2. 在确认对话框中点击 **"确定"**
3. ✅ 看到"删除成功"提示并自动刷新

## 💡 常用操作

### 筛选成本记录

```
1. 选择项目：在"项目"下拉框中选择
2. 选择类型：在"成本类型"下拉框中选择
3. 选择日期：在"日期范围"中选择起止日期
4. 点击"搜索"按钮查询
5. 点击"重置"按钮清空筛选条件
```

### 查看财务概览

页面顶部的4个卡片显示：
- 💰 **总预算** - 所有项目的预算总额
- 💼 **总成本** - 已确认的成本总额
- 📈 **预期利润** - 总预算 - 总成本
- 🎁 **预估奖金** - 基于利润计算的奖金

### 导出数据

点击右上角的 **"导出数据"** 按钮（功能开发中）

## ⚠️ 注意事项

1. **金额限制**：最小0.01元，最大99,999,999.99元
2. **描述长度**：5-500个字符
3. **日期选择**：不能选择未来日期
4. **状态说明**：
   - 已确认：立即计入项目总成本
   - 待审核：需要审核后才计入

## 🔧 故障排除

### 问题：无法提交表单

**解决方法：**
1. 检查所有必填项是否已填写
2. 查看表单项下方是否有红色错误提示
3. 确保金额大于0
4. 确保描述长度在5-500字符之间

### 问题：编辑按钮点击无反应

**解决方法：**
1. 刷新页面重试
2. 检查是否有权限编辑
3. 打开浏览器控制台查看错误信息
4. 联系管理员

### 问题：删除后数据未消失

**解决方法：**
1. 刷新页面
2. 检查是否真的删除成功（看提示）
3. 清除浏览器缓存

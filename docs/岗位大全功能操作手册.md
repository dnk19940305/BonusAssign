## 二、功能模块概览

### 1. 岗位分类管理
- **数据规模**：8个岗位分类（含主分类和子分类）
- **核心功能**：分类增删改查、树形展示、状态管理

### 2. 技能标签管理
- **数据规模**：4个技能分类，16个技能标签

- **数据规模**：2个职业发展路径模板（P3、P4）
- **核心功能**：模板管理、模板复制、启用/禁用

## 三、岗位分类管理操作指南

### 3.1 访问入口
1. 使用 `hr_manager` 账号登录系统
2. 导航至：**系统管理 → 岗位大全 → 设置管理**
3. 点击 **岗位分类管理** 标签页

### 3.2 查看分类结构
**当前分类树**：
```
├── 技术研发 (TECH)
│   ├── 前端开发 (TECH_FE)
│   └── 后端开发 (TECH_BE)
├── 产品设计 (PRODUCT)
├── 视觉设计 (DESIGN)
├── 运营市场 (OPERATION)
├── 职能支持 (SUPPORT)
└── 管理岗位 (MANAGEMENT)
```

### 3.3 新增分类
1. 点击 **新增分类** 按钮
2. 填写分类信息：
   - **分类名称**：必填，如"数据分析"
   - **分类代码**：必填，唯一标识，如"DATA"
   - **分类类型**：主分类(main) 或 子分类(sub)
   - **上级分类**：选择子分类时必填
   - **排序**：数字越小越靠前
   - **描述**：可选
   - **图标**：Element Plus图标类名
   - **颜色**：十六进制颜色值，如 #409eff
3. 点击 **确定** 保存

### 3.4 编辑分类
1. 在分类树中找到目标分类
2. 点击操作列的 **编辑** 按钮
3. 修改分类信息
4. 点击 **确定** 保存

### 3.5 删除分类
1. 点击目标分类的 **删除** 按钮
2. 确认删除操作
3. **注意**：
   - 有子分类的主分类无法删除
   - 关联岗位的分类无法删除

### 3.6 启用/禁用分类
- 点击分类右侧的 **状态开关** 切换启用状态
- 禁用后该分类不会在岗位选择中显示

### 3.7 刷新数据
- 点击右上角 **刷新** 按钮同步最新数据

## 四、技能标签管理操作指南

### 4.1 访问入口
1. 使用 `hr_manager` 账号登录
2. 导航至：**系统管理 → 岗位大全 → 设置管理**
3. 点击 **技能标签管理** 标签页

### 4.2 技能分类与标签概览

**技能分类体系**：
```
├── 技术技能 (6个标签)
│   ├── 前端开发、后端开发
│   ├── Vue.js、React
│   └── Node.js、Java
├── 软技能 (4个标签)
│   ├── 团队协作、沟通能力
│   └── 领导力、创新思维
├── 领域知识 (3个标签)
│   ├── 金融知识、电商业务
│   └── 教育行业
└── 管理能力 (3个标签)
    ├── 项目管理、规划能力
    └── 执行力
```

### 4.3 技能分类管理

#### 4.3.1 新增分类
1. 点击 **新增分类** 按钮
2. 填写：
   - **分类名称**：必填
   - **分类描述**：可选
   - **颜色标识**：用于前端展示
   - **排序权重**：控制显示顺序
3. 点击 **确定**

#### 4.3.2 编辑/删除分类
- **编辑**：点击分类操作列的编辑按钮
- **删除**：仅可删除无关联标签的分类

### 4.4 技能标签管理

#### 4.4.1 查看标签列表
- 顶部显示技能分类统计卡片
- 主表格展示所有标签
- 支持按分类、级别、状态筛选
- 支持关键词搜索（标签名称、代码、描述）

#### 4.4.2 新增标签
1. 点击 **新增标签** 按钮
2. 填写标签信息：
   - **标签名称**：必填，如"Python开发"
   - **标签代码**：必填，唯一标识，如"PYTHON"
   - **技能分类**：必填，选择所属分类
   - **技能级别**：必填，可选值：
     - `basic`：基础
     - `intermediate`：中级
     - `advanced`：高级
     - `expert`：专家
   - **标签描述**：可选
   - **同义词**：可选，多个用逗号分隔
   - **关联技能**：可选，选择相关的技能标签ID
   - **是否启用**：默认启用
3. 点击 **确定**

#### 4.4.3 编辑标签
1. 点击标签操作列的 **编辑** 按钮
2. 修改标签信息
3. 点击 **确定** 保存

#### 4.4.4 批量操作
1. 勾选多个标签（支持多选）
2. 点击表格上方的批量操作按钮：
   - **批量启用**：将选中的标签设为启用状态
   - **批量禁用**：将选中的标签设为禁用状态
   - **批量删除**：删除选中的标签（需确认）

#### 4.4.5 合并标签
**使用场景**：将重复或相似的标签合并为一个
1. 勾选需要合并的多个标签（至少2个）
2. 点击 **合并标签** 按钮
3. 在弹窗中选择：
   - **保留标签**：合并后保留的目标标签
   - **合并操作**：确认后将其他标签的引用迁移到保留标签
4. 点击 **确定合并**

#### 4.4.6 查看标签详情
- 点击标签行的 **详情** 按钮
- 查看完整信息：同义词列表、关联技能、使用统计

### 4.5 标签搜索与筛选
1. **分类筛选**：顶部下拉选择框，快速筛选某个分类的标签
2. **级别筛选**：选择技能级别
3. **状态筛选**：启用/禁用
4. **关键词搜索**：输入框支持模糊搜索标签名称、代码、描述

## 五、职业路径模板管理操作指南

### 5.1 访问入口
1. 使用 `hr_manager` 账号登录
2. 导航至：**系统管理 → 岗位大全 → 设置管理**
3. 点击 **职业路径模板** 标签页

### 5.2 当前模板概览

**P3技术工程师发展路径**：
- **职级**：P3
- **分类**：技术类(technical)
- **下一级别**：P4高级工程师
- **预计时间**：18-24个月
- **状态**：启用中（默认模板）

**P4高级工程师发展路径**：
- **职级**：P4
- **分类**：技术类(technical)
- **下一级别**：P5技术专家
- **预计时间**：24-36个月
- **状态**：启用中（默认模板）

### 5.3 新增模板
1. 点击 **新增模板** 按钮
2. 填写基本信息：
   - **模板名称**：必填，如"P5技术专家发展路径"
   - **模板代码**：必填，唯一标识，如"P5_TECH_PATH"
   - **职级**：必填，如P5
   - **分类**：必填，选择技术类/管理类/业务类等
   - **模板描述**：可选
   - **版本号**：默认1.0.0
   - **状态**：草稿(draft)/启用(active)/停用(inactive)
   - **是否默认模板**：是否作为该职级的默认路径

3. 填写职业发展规划：
   - **下一级别**：如"P6资深专家"
   - **预计时间**：如"36-48个月"
   - **横向发展**：可选的转岗方向（JSON数组）
   - **专业化方向**：细分专业领域（JSON数组）
   - **成长领域**：重点提升方向（JSON数组）

4. 填写晋升要求：
   - **最低工作年限**：如"6年以上相关工作经验"
   - **技能评估要求**：如"精通技术栈，具备系统架构能力"
   - **项目贡献要求**：如"主导大型项目，攻克核心难题"
   - **绩效要求**：如"绩效达到卓越"

5. 填写技能发展：
   - **核心技能**：必备技能（JSON数组）
   - **进阶技能**：高级能力（JSON数组）
   - **领导力技能**：管理能力（JSON数组）
   - **培训课程**：推荐课程（JSON数组）
   - **认证要求**：推荐认证（JSON数组）
   - **实践项目**：推荐项目类型（JSON数组）

6. 点击 **确定** 创建

### 5.4 编辑模板
1. 点击模板操作列的 **编辑** 按钮
2. 修改模板内容
3. 点击 **确定** 保存

### 5.5 复制模板
**使用场景**：基于现有模板创建新模板
1. 点击模板操作列的 **复制** 按钮
2. 系统自动创建副本（名称后缀"-副本"）
3. 编辑副本内容
4. 保存新模板

### 5.6 启用/停用模板
- 点击模板操作列的 **启用/停用** 按钮
- 停用的模板不会在员工职业规划中显示

### 5.7 删除模板
1. 点击模板操作列的 **删除** 按钮
2. 确认删除操作
3. **注意**：已被员工使用的模板无法删除

### 5.8 查看模板详情
- 点击模板名称或 **查看详情** 按钮
- 展示完整的职业发展路径规划
- 包含所有晋升要求和技能发展内容

## 六、常见操作场景

### 6.1 新增一个技术岗位的完整流程
1. **创建岗位分类**（如已存在则跳过）
   - 进入岗位分类管理
   - 新增"大数据"分类，上级选择"技术研发"

2. **创建相关技能标签**
   - 进入技能标签管理
   - 在"技术技能"分类下新增：
     - Spark开发（advanced）
     - Hadoop（intermediate）
     - 数据挖掘（advanced）

3. **创建职业发展路径**
   - 进入职业路径模板管理
   - 新增"P3大数据工程师发展路径"
   - 关联核心技能：Spark、Hadoop、数据挖掘

### 6.2 批量更新技能标签状态
1. 进入技能标签管理
2. 使用筛选条件找到目标标签
3. 勾选多个标签
4. 点击"批量启用"或"批量禁用"

### 6.3 合并重复的技能标签
**示例**：合并"前端开发"和"FE开发"
1. 进入技能标签管理
2. 勾选"前端开发"和"FE开发"
3. 点击"合并标签"
4. 选择保留"前端开发"
5. 确认合并（"FE开发"的所有引用将迁移到"前端开发"）

### 6.4 设置默认职业路径模板
1. 进入职业路径模板管理
2. 编辑目标模板
3. 勾选"设为默认模板"
4. **注意**：同一职级同一分类只能有一个默认模板
5. 保存后系统自动取消其他模板的默认状态

## 八、注意事项

### 8.1 数据安全
- ⚠️ 删除操作不可恢复，请谨慎操作
- ⚠️ 有关联数据的记录无法删除
- ⚠️ 合并标签会将所有引用迁移，请确认后操作

### 8.2 权限说明
- **HR管理员**：拥有所有管理权限
- **项目经理**：可查看，无法修改
- **普通用户**：可查看岗位和技能信息

### 8.3 数据一致性
- 代码字段必须唯一，系统会自动校验
- 分类层级最多2级
- JSON数组字段格式必须正确

### 8.4 性能优化
- 标签列表支持分页，默认每页20条
- 大量数据时建议使用筛选条件
- 批量操作单次不超过100条
